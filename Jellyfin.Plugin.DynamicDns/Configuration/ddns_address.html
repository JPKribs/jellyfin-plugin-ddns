Address

Updates your public DNS record when your server's public IP address changes. This runs on a schedule and updates your provider over outbound HTTPS. Choose which IP versions to maintain and how the public IP is detected. Set the update interval in Scheduled Tasks under Update Dynamic DNS.
Every endpoint used to detect your public IPv4, one per line, tried in order until one returns a valid address. Add or remove any. Leave blank to use the built in defaults.
Every endpoint used to detect your public IPv6, one per line, tried in order until one returns a valid address. Add or remove any. Leave blank to use the built in defaults.
An address is treated as internal when it falls in a private, loopback, link local, CGNAT, or other reserved range. Disable this setting if your IP is being misclassified as internal.
Submits your IP to your provider even when DNS already matches. Best used for proxied DNS entries where only local lookups can occur to prevent duplicate updates.
Consecutive failures for a record before it's paused. This prevents a wrong configuration from spamming a provider and risking a ban.
How long a paused record waits before it is tried again. If it fails again it backs off for the same period.
How long each detection or provider request may take before it is abandoned.
