# NearlyFreeSpeech.NET

Set the record's **Provider** to **NearlyFreeSpeech.NET**, then fill these fields.

- **Hostname**: the host. Only the A (IPv4) record is managed, matching ddclient.
- **Login**: your NFSN member login.
- **Password**: your NFSN API key. Request one on the NFSN site under Profile then API.
- **Zone**: the DNS zone, such as example.com.

_Ported from ddclient's `nic_nfsn_update`._
