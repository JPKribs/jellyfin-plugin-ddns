# DynDNS v2

Set the record's **Provider** to **DynDNS v2**, then fill these fields.

- **Hostname**: the fully qualified host.
- **Login**: your account username, or the update token, at the DynDNS v2 compatible service.
- **Password**: your account password or token.
- **Server**: the update server host. Defaults to members.dyndns.org. Set it to your provider's update host.

_Ported from ddclient's `nic_dyndns2_update`._
