# Mythic Beasts

Set the record's **Provider** to **Mythic Beasts**, then fill these fields.

- **Hostname**: the record name. The service sets it to the request's source address.
- **Login**: your Mythic Beasts dynamic DNS API key.
- **Password**: the API key secret. Create a Dynamic DNS credential in the Mythic Beasts panel.

_Ported from ddclient's `nic_mythicdyn_update`._
