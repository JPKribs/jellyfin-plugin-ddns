# Hurricane Electric (dns.he.net)

Set the record's **Provider** to **Hurricane Electric (dns.he.net)**, then fill these fields.

- **Hostname**: the record name. It is also used as the auth user.
- **Password**: the DDNS key generated for that record. In the dns.he.net panel, enable dynamic DNS on the A or AAAA record and copy the generated key.

_Ported from ddclient's `nic_henet_update`._
