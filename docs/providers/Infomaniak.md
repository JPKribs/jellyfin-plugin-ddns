# Infomaniak

Set the record's **Provider** to **Infomaniak**, then fill these fields.

- **Hostname**: the record.
- **Login**: your Infomaniak dynamic DNS username.
- **Password**: your Infomaniak dynamic DNS password. Create dynamic DNS credentials for the domain in the Infomaniak Manager.

_Ported from ddclient's `nic_infomaniak_update`._
