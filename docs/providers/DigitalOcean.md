# DigitalOcean

Set the record's **Provider** to **DigitalOcean**, then fill these fields.

- **Hostname**: the record name, such as home.example.com.
- **Password**: a DigitalOcean API token with read and write scope (API then Tokens then Generate New Token).
- **Zone**: the apex domain, such as example.com.

_Ported from ddclient's `nic_digitalocean_update`._
