# Yandex 360 / PDD

Set the record's **Provider** to **Yandex 360 / PDD**, then fill these fields.

- **Hostname**: the fully qualified host to match.
- **Login**: your Yandex domain. It is used to list the domain's records.
- **Password**: your Yandex PDD API token, sent as the PddToken header. Get it from the Yandex PDD or Connect admin.

_Ported from ddclient's `nic_yandex_update`._
