# ClouDNS

Set the record's **Provider** to **ClouDNS**, then fill these fields.

- **Hostname**: the record you enabled a Dynamic URL on.
- **Password**: the Dynamic URL (DynURL) for the record. In the ClouDNS panel open the A or AAAA record, enable Dynamic URL, and paste the full URL here.

_Ported from ddclient's `nic_cloudns_update`._
